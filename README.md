# 🟣 Blurple Pro Calculator & Converter

A sleek, mobile-first web application that combines a dual-logic mathematical calculator with a smart, dynamic unit and currency converter. Built with vanilla web technologies, it features a beautiful "Blurple" dark mode UI and works perfectly as an installable Progressive Web App (PWA).

---

## ✨ Key Features

### 🧮 Dual-Logic Calculator
* **BODMAS Mode:** Evaluates complex expressions using standard mathematical order of operations (e.g., handles scientific functions like `sin`, `cos`, and `√`).
* **Sequential Mode:** Calculates strictly from left to right, mimicking traditional basic pocket calculators.
* **Infinite History Log:** Automatically saves all your calculations to local storage so you never lose your work.

### 🔄 Smart Converter
* **Dynamic Dropdowns:** Intelligently filters options so you can only convert compatible units (e.g., Length to Length, Data to Data).
* **Live Currency Rates:** Fetches real-time exchange rates via the ER-API.
* **Offline Support:** Saves the latest currency rates and your previous conversions to your device. If you lose internet, the app seamlessly switches to offline mode with web toast notifications!

---

## 🛠️ Built With

* **HTML5:** Semantic structure and PWA meta tags.
* **CSS3:** Custom Flexbox and Grid layouts, smooth animations, and a modern #5865F2 (Blurple) theme.
* **JavaScript (ES6):** Vanilla JS handling the DOM manipulation, math parsing, `localStorage`, and asynchronous API fetching.

---

## 📱 How to Install on Mobile

If you choose to host this code on a secure web server (like GitHub Pages) in the future, it functions as a Progressive Web App that can be installed directly to a phone:

1. Open the hosted link in **Chrome** (Android) or **Safari** (iOS).
2. Tap the browser's menu (three dots in Chrome, or the Share button in Safari).
3. Select **"Add to Home Screen"**.
4. The app will now appear on your home screen and open in a full-screen, native-like experience!

---

## 💻 How to Run Locally

To run this app on your own computer or device:

1. Clone this repository or download the `.zip` file.
2. Extract the folder.
3. Open the `index.html` file in any modern web browser.
4. No servers, build steps, or dependencies required!
